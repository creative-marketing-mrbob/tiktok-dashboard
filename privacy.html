Privacy Policy

This TikTok Analytics Dashboard accesses the following TikTok user data via the TikTok Display API:

  • Display name and profile information (user.info.basic)
  • Video list and performance metrics (video.list)

This data is used exclusively to display analytics to the authenticated account owner. No data is stored on external servers, sold, or shared with any third party. Access tokens are stored locally in a .env file on the user's own machine.
